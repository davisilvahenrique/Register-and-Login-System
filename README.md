## Descrição e Motivo do projeto:
- Sistema de Login e Registro feito para treino.
- Site responsivo! (Funciona em desktop e celular)

### Front-end
- HTML5
- CSS3

### Back-end
- JS
- localStorage do proprio navegador para salvar os registros
